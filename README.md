## 📋 Описание

Этот репозиторий содержит **учебную практику по C#**, выполненную в рамках обучения в **ZAMT**.

В рамках практики необходимо было реализовать **аналоги классических игр**, используя **Windows Forms или WPF** и язык программирования **C#**.

Основная цель - изучение:

* работы с **графическим интерфейсом WinForms**;
* обработки **клавиатурного ввода**;
* реализации **игровой логики**.

## 🐍 Snake (Windows Forms)

Реализация классической игры **Snake** с использованием **Windows Forms**.

Функциональность игры:

* управление змейкой с клавиатуры;
* генерация еды;
* увеличение длины змейки;
* проверка столкновений;
* разные уровни сложности;
* пауза в любой момент.

Игровая логика реализована на **C#**, интерфейс - через **WinForms**.

## 🚀 Запуск проекта

1. Перейти в раздел **Releases** репозитория
2. Скачать zip архив с последней версией программы
3. Распаковать архив
4. Запустить исполняемый файл:

```text
Snake.exe
```
